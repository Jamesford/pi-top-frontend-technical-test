# pi-top - Frontend Technical Test

### Install
```shell
yarn install
```

### Run Development

Build on change with live reloading / HMR
```
yarn start
```

Run tests
```
yarn run test
```

Watch tests
```
yarn run test:watch
```

### Run Production

Build app for production
```
yarn run build
```

Serve production locally
```
npx serve dist
```
